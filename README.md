# Scrap Books–Film Metadata Project

## ภาพรวมโครงการ (Project Overview)
โครงการนี้มีวัตถุประสงค์เพื่อรวบรวมและเชื่อมโยงข้อมูลเชิงเมตาดาตา (metadata)  
ระหว่าง **หนังสือ** และ **ภาพยนตร์** โดยมุ่งเน้นไปที่กรณีของ  
> *ภาพยนตร์ที่ถูกดัดแปลงมาจากหนังสือ (Book-to-Film Adaptations)*

ข้อมูลถูกนำมาจากหลายแหล่ง และถูกใช้เป็นตัวกลางในการเชื่อมความสัมพันธ์  
ระหว่าง metadata ของหนังสือและภาพยนตร์ เพื่อรองรับการวิเคราะห์เชิงข้อมูลในลำดับถัดไป

---

## การเลือกแหล่งข้อมูล (Data Sources)

### 1. Kaggle Dataset
ใช้ชุดข้อมูลจาก Kaggle ซึ่งเป็นแหล่งข้อมูลเปิดที่ได้รับความนิยม  
โดยข้อมูลหลักประกอบด้วย metadata ของภาพยนตร์ที่รวบรวมมาจาก **IMDb**

- จำนวนภาพยนตร์ทั้งหมด: **45,466 เรื่อง**
- ตัวอย่างข้อมูลที่มี:
  - ชื่อภาพยนตร์
  - ปีที่ออกฉาย
  - ประเภท (Genre)
  - คะแนน (Rating)
  - รายละเอียดเชิง metadata อื่น ๆ

ชุดข้อมูลนี้ทำหน้าที่เป็นฐานข้อมูลหลักของฝั่งภาพยนตร์

---

### 2. Wikipedia
ใช้ Wikipedia เป็นแหล่งข้อมูลเสริม  
เพื่อค้นหาและยืนยันความสัมพันธ์ระหว่าง **หนังสือ** และ **ภาพยนตร์ที่ถูกดัดแปลงมาจากหนังสือ**

Wikipedia ทำหน้าที่เป็น:
- แหล่งอ้างอิงความสัมพันธ์ (reference layer)
- ตัวกลางในการตรวจสอบว่าเรื่องใดเป็นการดัดแปลงจากวรรณกรรม

---

### 3. Goodreads
ใช้ข้อมูลหนังสือยอดนิยมจากเว็บไซต์ **Goodreads**

- จำนวนหนังสือทั้งหมด: **5,000 เรื่อง**
- ข้อมูลที่เก็บรวบรวม:
  - ชื่อหนังสือ
  - ผู้เขียน
  - คะแนนความนิยม
  - รายละเอียด metadata ของหนังสือ

ข้อมูลจาก Goodreads ถูกนำมาใช้เพื่อ:
- ค้นหาหนังสือที่ถูกนำไปดัดแปลงเป็นภาพยนตร์
- เชื่อมโยง metadata ระหว่างฝั่งหนังสือและภาพยนตร์

---

## แนวคิดการเชื่อมโยงข้อมูล (Data Linking Concept)

โครงการนี้ใช้แนวคิด **หนังสือเป็นตัวกลาง (Book-Centric Linking)**  
เพื่อเชื่อมข้อมูลระหว่างสองโดเมนหลัก ได้แก่

- metadata ของหนังสือ (Goodreads)
- metadata ของภาพยนตร์ (IMDb / Kaggle)

กระบวนการโดยสรุป:
1. รวบรวม metadata ของหนังสือจาก Goodreads
2. รวบรวม metadata ของภาพยนตร์จาก Kaggle (IMDb)
3. ใช้ข้อมูลจาก Wikipedia เพื่อตรวจสอบความสัมพันธ์
4. สร้างชุดข้อมูล metadata ของภาพยนตร์ที่ถูกดัดแปลงจากหนังสือ

ผลลัพธ์ที่ได้สามารถนำไปใช้ต่อในงาน:
- Data Analysis
- Recommendation System
- Network Analysis ระหว่างวรรณกรรมและภาพยนตร์
- งานวิจัยด้านสื่อและวัฒนธรรม

## โครงสร้างข้อมูล (Data Artifacts)

เช่น:
- `goodreads_metadata.json`
- `movies_metadata.json`
- `books_into_films_metadata.json`
- `all_books_into_films.json`
